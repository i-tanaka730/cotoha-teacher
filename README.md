# cotoha-teacher

## Overview

This is a LINE BOT that uses the sentiment analysis of the COTOHA API to check whether messages entered on LINE contain negative words.

## Requirement

- Python 3.8.1
- COTOHA API
- LINE Messaging API
- Heroku
